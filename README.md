Apartman Yönetim Sistemi (AYS) / Apartment Management System

Bu platform; apartman ve site yönetim süreçlerini bütünüyle dijitalleştirmek, yönetim ile kat sakinleri arasındaki finansal akışı ve iletişimi optimize etmek amacıyla geliştirilmiş uçtan uca bir **Web Otomasyonudur**. Mimari olarak **ASP.NET Core MVC** deseni üzerine kurulmuş olup, katmanlı yazılım prensiplerine uygun olarak inşa edilmiştir.

#### Öne Çıkan Teknik Özellikler:

* **Çift Yönlü Rol Yapısı:** Yönetici (Admin) ve Kat Sakini (User) için birbirinden tamamen bağımsız iki ayrı operasyonel panel sunar.
* **Asenkron Kimlik Doğrulama:** JavaScript Fetch API mimarisi kullanılarak, sayfa yenilenmeden (AJAX) asenkron çalışan giriş ve kayıt mekanizmaları barındırır.
* **Dinamik Finans & Aidat Yönetimi:** Yönetim tarafından toplu veya hücre bazlı borçlandırma yapılabilmesini; sakinlerin ise borçlarını anlık takip edip durum güncellemesi (Ödeme Simülasyonu) tetikleyebilmesini sağlar.
* **Çift Yönlü Talep Motoru:** Sakinlerin oluşturduğu taleplerin, yönetim panelindeki asenkron pop-up (modal) pencereleri üzerinden anlık okunup yanıtlanabilmesine imkan tanır.
* **Veri Güvenliği Duvarı:** Kritik veri silme operasyonlarında yanlışlıkları önlemek adına SweetAlert2 kütüphanesiyle entegre onay mekanizmaları işletir.

#### Teknolojiler :

* **Backend:** C#, ASP.NET Core MVC
* **ORM & Veri Katmanı:** Entity Framework Core (Code-First), LINQ, SQL Server
* **Frontend:** HTML5, CSS3 (Custom Grid & Flexbox, Koyu/Açık Tema Desteği), JavaScript (ES6+, Fetch API, Async/Await)
* **Dış Kütüphaneler:** FontAwesome, Google Fonts, SweetAlert2

### Overview (English)

This platform is an end-to-end **Web Automation System** designed to fully digitalize apartment and complex management workflows, optimizing financial tracking and communication between the administration and residents. Architecturally, it is built on top of the **ASP.NET Core MVC** pattern, following layered software engineering principles.

####  Key Technical Features:

* **Dual-Role Architecture:** Provides two separate operational panels for the Property Manager (Admin) and Apartment Residents (User).
* **Asynchronous Authentication:** Implements seamless, non-reloading login and registration mechanisms (AJAX) powered by the JavaScript Fetch API.
* **Dynamic Finance & Dues Management:** Enables the administration to enforce bulk or unit-based billing, while allowing residents to track debts in real-time and trigger status updates (Payment Simulation).
* **Bidirectional Request Engine:** Allows resident-submitted tickets to be reviewed and answered instantly via asynchronous pop-up (modal) windows on the admin dashboard.
* **Data Safety Layer:** Restricts accidental data loss during critical deletion operations by deploying approval workflows integrated with SweetAlert2.

####  Technology Stack:

* **Backend:** C#, ASP.NET Core MVC
* **ORM & Data Layer:** Entity Framework Core (Code-First), LINQ, SQL Server
* **Frontend:** HTML5, CSS3 (Custom Grid & Flexbox, Dark/Light Theme Support), JavaScript (ES6+, Fetch API, Async/Await)
* **Third-Party Libraries:** FontAwesome, Google Fonts, SweetAlert2
